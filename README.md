# TarkovMapTracker

Автоматическая передача координат из игры Escape from Tarkov на карту tarkov-market.ru/maps без сворачивания игры и потери фокуса.

---

## Как это работает

Проект состоит из двух частей, которые общаются через локальный HTTP-сервер на 127.0.0.1:12345.

```
Escape from Tarkov          TarkovMapTracker.py          tarkov-market.ru/maps
(мод скриншотов с коорд.)   (локальный сервер)           (карта в браузере)

[Скриншот] сохраняет        Watchdog следит за папкой    UserScript опрашивает
PNG с именем-координатой    Screenshots, читает имя,     сервер раз в секунду,
                            удаляет файл, отдаёт по      вставляет координаты
                            запросу как JSON             в поле ввода и жмёт Enter
```

Поток данных:

1. В игре игрок делает скриншот (например, по F12 или горячей клавише мода). Мод сохраняет PNG-файл с именем, содержащим координаты.
2. Python-сервер через библиотеку watchdog замечает новый файл в папке `Documents/Escape from Tarkov/Screenshots`.
3. Скрипт берёт имя файла (без расширения), сохраняет его в переменную `latest_coords` и удаляет сам PNG, чтобы папка не засорялась.
4. Userscript в браузере раз в секунду делает GET-запрос на `http://127.0.0.1:12345/`.
5. Сервер отдаёт JSON вида `{"coords": "x_123_y_45_z_56"}` и сразу очищает значение (одно сообщение — один запрос).
6. UserScript находит на странице карты `input[placeholder*="Вставьте сюда имя файла"]`, подставляет туда координаты, симулирует события `input`, `change` и `keydown(Enter)`.
7. Карта переключается в нужную точку. Игрок даже не вышел из игры.

---

## Возможности

- Полностью автоматический поток: от скриншота до обновления карты примерно за одну секунду.
- Без потери фокуса игры — игра не сворачивается, мышь не улетает на второй монитор.
- Автоочистка скриншотов — PNG-файлы удаляются сразу после чтения координат, папка не раздувается.
- Лёгкий локальный сервер без сторонних облачных сервисов, всё крутится на 127.0.0.1.
- CORS настроен — Userscript ходит на локальный сервер без ошибок политики безопасности.
- Автообновление UserScript — Tampermonkey сам подтягивает новую версию с GitHub при наличии `@updateURL`.
- Минимум зависимостей — нужен только Python и один pip-пакет watchdog.

## Требования

| Компонент | Версия / значение |
|---|---|
| ОС | Windows (путь к скриншотам захардкожен под Documents) |
| Python | 3.8+ |

## Установка

### Шаг 1. Клонирование репозитория

```bash
git clone https://github.com/Onzis/TarkovMapTracker.git
cd TarkovMapTracker
```

Либо скачайте ZIP и распакуйте в любую папку.

### Шаг 2. Установка Python-зависимостей

```bash
pip install watchdog
```

Если `pip` ругается на права, используйте `pip install --user watchdog` или активируйте виртуальное окружение.

### Шаг 3. Установка UserScript

1. Установите расширение Tampermonkey в браузер.
2. Откройте файл `TarkovMapTrackerAuto.user.js` на GitHub в режиме Raw (кнопка Raw справа вверху над кодом файла).
3. Tampermonkey автоматически предложит установить скрипт — нажмите «Установить».
4. Убедитесь, что скрипт включён в панели Tampermonkey.

## Использование

### 1. Запуск Python-сервера

Откройте терминал (cmd или PowerShell) и выполните:

```bash
python TarkovMapTracker.py
```

При успешном запуске вы увидите:

```
Сервер запущен. Мониторинг папки: C:\Users\<Имя>\Documents\Escape from Tarkov\Screenshots
```

Окно терминала не закрывайте — пусть висит в фоне, пока играете.

### 2. Откройте карту

В браузере откройте нужную карту на `https://tarkov-market.ru/maps/...`. Вкладку можно держать на втором мониторе или свернуть.

### 3. Запустите Escape from Tarkov

Зайдите в рейд. Когда нужно узнать свои координаты — сделайте скриншот клавишей, которую назначил ваш мод. Через секунду на карте автоматически появится маркер.

### 4. Завершение работы

Чтобы остановить сервер, в окне терминала нажмите `Ctrl+C`.

## Конфигурация

По умолчанию скрипт рассчитан на стандартную установку Escape from Tarkov на Windows. Если у вас нестандартная ситуация — правьте константы в начале `TarkovMapTracker.py`.

### Папка скриншотов

```python
WATCH_DIR = os.path.join(USER_HOME, "Documents", "Escape from Tarkov", "Screenshots")
```

Если игра установлена на другой диск или папка Documents перенесена, измените `WATCH_DIR` вручную.

### Порт сервера

```python
server = HTTPServer(('127.0.0.1', 12345), CoordinatesServer)
```

Если порт 12345 занят другим приложением, поменяйте его одновременно в `TarkovMapTracker.py` и в `TarkovMapTrackerAuto.user.js` (в строке `url: "http://127.0.0.1:12345/"`).

### Частота опроса сервера браузером

В `TarkovMapTrackerAuto.user.js`:

```javascript
}, 1000); // Частота проверки — 1 раз в секунду
```

Можно уменьшить до 500 (полсекунды) для более быстрого отклика или увеличить до 2000–3000 для экономии ресурсов.

## Решение проблем

### «Путь не найден: .../Escape from Tarkov/Screenshots»

Скрипт не нашёл папку. Создайте её вручную или проверьте, что Escape from Tarkov действительно сохраняет скриншоты туда. Альтернативно — укажите правильный путь в `WATCH_DIR`.

### Координаты не появляются на карте

Проверьте по порядку:

1. Запущен ли Python-сервер. В терминале должно быть сообщение «Сервер запущен».
2. Активен ли UserScript. Откройте панель Tampermonkey и убедитесь, что TarkovMapTrackerAuto включён.
3. Открыта ли правильная страница. URL должен начинаться с `https://tarkov-market.ru/maps/`. На главной `tarkov-market.ru` скрипт не сработает — нужен именно раздел `/maps/*`.
4. Создаётся ли скриншот в нужной папке. Зайдите в `Documents/Escape from Tarkov/Screenshots` и сделайте тестовый скриншот. Если PNG не появляется — проблема в моде или настройках игры, а не в TarkovMapTracker.
5. Не блокирует ли брандмауэр порт 12345. Хотя сервер слушает только 127.0.0.1, иногда антивирусы (Kaspersky, ESET) перехватывают локальные запросы. Добавьте Python в исключения.

### Скриншоты удаляются, но координаты не уходят

Проверьте, что имя файла действительно содержит координаты в формате, который понимает сайт tarkov-market.ru. Скрипт передаёт имя как есть, без обработки.

### Консоль браузера показывает ошибки CORS

В UserScript должны быть указаны:

```javascript
// @grant        GM_xmlhttpRequest
// @connect      127.0.0.1
```

Если этих строк нет — Tampermonkey не даст скрипту сделать запрос к локальному серверу. Убедитесь, что установлен свежий `TarkovMapTrackerAuto.user.js` из этого репозитория.

### Tampermonkey не предлагает установку скрипта

Откройте файл именно в Raw-режиме: на GitHub нажмите кнопку Raw над содержимым файла. URL должен заканчиваться на `.user.js`.

## Дисклеймер

- Соблюдайте правила игры. Использование любых сторонних утилит, чит-клиентов и модификаций Escape from Tarkov может нарушать EULA и привести к блокировке аккаунта. TarkovMapTracker сам по себе не взаимодействует с процессом игры — он только читает файлы скриншотов из папки Documents и работает с браузером. Используйте на свой страх и риск.
- Скриншоты удаляются. После обработки координат PNG-файл удаляется безвозвратно. Если нужно сохранять скриншоты для истории — настройте отдельную папку или отключите строку `os.remove(file_path)` в `TarkovMapTracker.py`.
- Приватность. Сервер слушает только 127.0.0.1 (localhost) — он недоступен из интернета и локальной сети. Никакие данные никуда не отправляются.
